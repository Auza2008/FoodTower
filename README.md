## 配置开发环境
1.首先你要有一个[Gradle](https://gradle.org/)和[Java8[JDK]](https://www.oracle.com/cn/java/technologies/downloads)

2.打开你的idea

3.选择src并导入

4.等待下载依赖库完成，然后运行```gradlew idea build```
